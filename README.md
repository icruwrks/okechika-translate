# okechika-translate

## 概要

- 桶地下のサイトを解析するための個人プロジェクトです
- 解析対象のサイト: https://www.qtes9gu0k.xyz

## 使い方
- taiouhyou.csvを元に、index.htmlを翻訳します
- 翻訳後は〇〇_translate.htmlになります

## コマンド

```
python .\translate_html_v2.py .\あげぎぺやほせ.html
python .\translate_html_v2.py '.\倏俊丣亞乱丌代 _ あげぎぺやほせ.html'
python .\translate_html_v2.py '.\什儺伏丂佄代伸 _ あげぎぺやほせ.html'
```
